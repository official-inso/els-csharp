# Базовый пример (RU)

Консольное приложение, демонстрирующее основные возможности SDK: асинхронный
захват, синхронную отправку критических ошибок, проверку доступности сервера
и корректное завершение.

```bash
export ELS_ENDPOINT=https://api.insoweb.ru/els
export ELS_API_KEY=ваш-api-ключ
dotnet run --project examples/ru/Basic
```
