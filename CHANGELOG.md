### Added
- Initial C# SDK with async batching, retry logic, disk buffering.
- `Inso.Els` core package (target frameworks: netstandard2.0, netstandard2.1, net6.0, net8.0, net9.0).
- `Inso.Els.AspNetCore` for ASP.NET Core middleware and dependency injection.
- `Inso.Els.Extensions.Logging` for `Microsoft.Extensions.Logging` provider.
- Examples (EN/RU) for basic, ASP.NET Core middleware, and ILogger integration scenarios.
- Documentation (EN/RU): `README`, `docs/FIELDS`, `docs/ARCHITECTURE`.
